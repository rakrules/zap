How does this work?  ⓘ

The PIN finds each other: When both devices type the same 4-digit PIN and click Connect, they meet through a tiny free public WebSocket service for about 2 seconds. They swap their network addresses, then disconnect.

Files go peer-to-peer: Actual transfers use WebRTC — a direct device-to-device connection. Files never touch any server, never get uploaded anywhere. No size limit. As fast as your WiFi allows.

Two-way: Either device can send files at any time. Whoever drops files is the sender for that batch. Switch roles freely.

Privacy note: Anyone who types the same 4-digit PIN at the same time would also connect. So pick a fresh random PIN each session (the ↻ button does this).
